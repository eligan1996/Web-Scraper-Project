# Python Job Listings Scraper

## Project
This project is part of the Roadmap.sh project series.
https://roadmap.sh/projects/job-listings-scraper

## Description: 
In this project, a Python web scraper collected job listings from a Fake Python jobs website. Information extracted were job title, company name, location, and a link to the full job description.   The results were stored in a CSV file.  Link to the site:
https://realpython.github.io/fake-jobs/  

## Requiredments:
Python 3
requests library
beautifulsoup4 library

# Installation:
Install the required dependencies using pip:  pip install requests beautifulsoup4.

## How to run:  
Navigate to the project folder and run the command below.
No installation required - Python 3 is all you need.

## Commands

python3 scraper.py
